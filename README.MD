# Recipe Finder
[![Build Status](https://travis-ci.com/ZintrulCre/Recipe-Finder.svg?branch=master)](https://travis-ci.com/ZintrulCre/Recipe-Finder)

## Introduction

- Input
    - items in fridge
    - recipes in json format
- Output
    - recommendation for what to cook

## Implementation

- Backend
    - Java 8
        - simplejson
    - Spring Boot
    - Maven

- Frontend
    - JavaScript
    - BootStrap 4
    - JQuery

- Deployment
    - Google Cloud Platform
        - Nginx
    - Travis CI
